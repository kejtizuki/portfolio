The daily view also offers an option to change the curve
rendering for the radial chart. Different curves are
displayed on the Figure \ref{fig:dailyLineType}. Starting
from the left, the chart is rendered using
d3.curveCardinalClosed, d3.curveBasisClosed and
d3.curveLinear. All of the methods of line interpolation
give a bit different results. As we needed the most
accurate approximation that would also look visually
appealing, the first interpolation type was chosen as the
default one in the final view. The users still have the
possibility to change it in the menu and explore which
option is the easiest to understand for them.
\begin{figure}
\centering
\includegraphics[width=1\textwidth]{figures/dailyRadial
    VsBar.png}
\caption{Daily data displayed in a form of radial chart
    and bar chart.}
\label{fig:dailyRadialVsBar}
\end{figure}
Daily view can be displayed in a form of a radial chart and
a bar chart. Both of the views have their pros and cons,
therefore it was decided that the users need to have an
option to switch between them. Radial chart was used as a
default visualization. The radial visualization assumes
that the daily patterns are continuous in contrast to the
bar chart which is a typical linear visualization. In this
case, the day starts and finishes.
Additionally, the radial visualization is integrated with
the times of sunset and sunrise. The two graphs show
exactly the same data measured on the same day, however the
careful viewers might notice slight differences between
these two graphs. For instance, between some hours in
radial graph the lines corresponding to average values are
very close to zero. In bar chart the same values are
usually close to one. This problem was not solved during
this project timeframe but it can be tackled in the future
work.
